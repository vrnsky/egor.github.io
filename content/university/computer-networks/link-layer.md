### Introduction to the link layer

Node examples:
- Host
- Router
- Switch

Link:
- Communication channel that connect adjacent nodes
- Examples:
  - Wired link
  - Wireless link

Frame:
- Layer 2 packet
- Encapsulated datagram

Data Link:
- Provide communication channel between two neighbor nodes to transfer frame
- Example: 802.11 (or WiFi)
